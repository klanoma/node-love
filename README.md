# Node Love

A Node.js boilerplate including:
- Node.js
- Express
- CoffeeScript
- Jade
- Compass
- Skeleton
- MongoDB
- Mocha


## Getting Started

- `npm install`
- `node app`
- To use compass, setup the compass rubygem (should probably use RVM) and run `compass watch`
