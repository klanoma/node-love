# Node Love

A Node.js boilerplate including:
- Node.js
- Express
- CoffeeScript
- Jade
- Compass
- Skeleton
- MongoDB
- Mocha


## Getting Started

- `npm install` (ensure mongodb is installed and running)
- `node server`
- `compass watch` (ensure compass is installed and running)
